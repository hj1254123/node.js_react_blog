## 文章文档

```json
[
  {
    "id": 1,
    "title": "文章标题",
    "time": 1539484051,
    "intro": "对文章的简介",
    "content": "文章内容（markdown格式）"
  },
  ...
]
```

> 文章简介非必选。没有设置自动截取文章第一段（200字数限制）。
