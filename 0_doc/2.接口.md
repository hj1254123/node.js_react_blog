## 鉴权

### 注册

路径：`/auth/register`

方法： `post`

参数：

```json
{
  "userName": "monkey",
  "password": "123123",
  "invitation_code": "xxx"
}
```

> 长度均为 6~16，不开放注册，采用邀请码机制。

返回示例：

```json
{
  "message": "注册成功",
  "data": {
    "id": 1,
    "userName": "monkey"
  }
}
```

### 登录

## 文章

### 添加文章（同时添加标签、简介）

路径：`/article`

方法：`post`

参数：

```json
{
  "title": "文章标题",
  "intro": "简介",
  "content": "文章内容",
  "tags": ["标签1", "标签2"],
  "token": "djskksd"
}
```

返回示例：

```json
{
  "message": "添加文章成功",
  "data": {
    "articleID": 1,
    "articleTime": 11111111,
    "title": "文章标题",
    "intro": "简介",
    "content": "文章内容",
    "tags": ["标签1", "标签2"]
  }
}
```

### 删除文章

路径：`/article`

方法：`delete`

参数：

```json
{
  "articleID": 1,
  "token": "djskksd"
}
```

返回示例：

```json
{
  "message": "文章删除成功",
  "data": {
     "articleID": 1
  }
}
```

### 修改文章：标题、简介、内容

路径：`/article`

方法：`put`

参数：

```json
{
  "articleID": 1,
  "title": "文章标题",
  "intro": "简介",
  "content": "文章内容",
  "token": "djskksd"
}
```

返回示例：

```json
{
  "message": "文章修改成功",
  "data": {
    "articleTime": 11111111,
    "articleID": 1,
    "title": "文章标题",
    "intro": "简介",
    "content": "文章内容",
    "tags": ["标签1", "标签2"]
  }
}
```

### 获取某篇文章

路径：`/article/:id`

方法：`get`

返回示例：

```json
{
  "message": "获取文章成功",
  "data": {
    "articleTime": 1540820691,
    "articleID": 1,
    "title": "文章标题",
    "content": "文章内容",
    "tags": [
      {"tagName": '标签1', "tagID": 1},
      {"tagName": '标签2', "tagID": 2},
    ]
  }
}
```

## 标签

### 向某篇文章追加标签

路径：`/tags`

方法：`post`

参数：

```json
{
  "articleID": 1,
  "tagName": "标签名称",
  "password": "410410"
}
```

返回示例：

```json
{
  "message": '追加标签成功',
  "data": {
    "articleID": 1,
    "tags": [
      {"tagName": '标签1', "tagID": 1},
      {"tagName": '标签2', "tagID": 2},
    ]
  }
}
```

		{

			"month": "一月",
			"articlesData": [
				{
           "articleTime": 1111111,

           "tags": [
            	{"tagName": '标签1', "tagID": 1},
            	{"tagName": '标签2', "tagID": 2},
          	]
				},
         ...
       ]
  	},
    ...（12个月/页）
